Aceasta aplicatie are ca scop de baza determinarea posibilitatii ajungerii dintr-o locatie în alta folosind locatiile si drumurile date.
Pentru a obtine aceasta informatie se foloseste un algoritm de parcurgere DFS care se afla in clasa DepthFirstSearch. 
In clasa Problem se fac verificari si adaugari de informatii ale unei instante de tipul problemei date. 
Mai multe detalii cu ceea ce face fiecare metoda din toate clasele create sunt disponibile in documentatia claselor generata de javadoc.
